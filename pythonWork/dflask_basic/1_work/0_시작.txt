* Flask
    ` 파이썬으로 만든 마이크로 앱 프레임워크
    ` 마이크로 앱 프레임워크 : 핵심 기능을 유지하면서 확장성을 가진 프레임워크
    ` 필요에 따라 다양한 확장 기능을 추가해서 개발

[참고] 파이썬 웹 프레임워크
	: Django / Flask / bottle / fastapi
1) Django
	` 파이썬 웹 프레임워크 중 가장 유명
	` 중규모 이상의 웹 앱을 구현하는데 자주 사용
	` 개발에 필요한 기능이 구현되어 있어서 풀스택 프레임워크로 불림
2) bottle 
	` 파이썬 웹 프레임워크 중 가장 단순
	` 간단한 앱을 만드는데 적합
	` 가볍고 빠름
3) FastAPI
	` 비동기 처리가 용이하도록 만들어진 파이썬 웹 프레임워크
	` 요청 처리 속도가 빠름
